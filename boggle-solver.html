Boggle Master Solver

July 2019

Project Description:

This program randomly generates a 5x5 Boggle Board, then solves for all the words in the board. A user can also input commands to run functions that show where specific words are located. The word list used to determine if a word is valid is a .txt file of the Scrabble dictionary.
